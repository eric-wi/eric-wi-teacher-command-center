## How to use

### Option 1: Google Sheets
1. Download this repository as a ZIP.
2. Unzip it.
3. Create a new Google Sheet workbook.
4. Import each CSV file as its own sheet.
5. Rename the tabs to match the CSV filenames without the numeric prefix.
6. Copy dashboard formulas from `docs/dashboard-formulas.md`.

### Option 2: Microsoft Excel
1. Download this repository as a ZIP.
2. Unzip it.
3. Open Excel.
4. Import each CSV as a separate worksheet into one workbook.
5. Apply formatting, filters, frozen panes, and charts.
6. Use the dashboard formulas guide to rebuild the dashboard cards and charts.
