# Floral Dashboard Visual Layout Plan

## Design direction
Create a calm, soft, primary-friendly teacher dashboard with a floral studio aesthetic.

## Colour palette
Use soft pastel tones.

### Core colours
- Blush Pink: `#F7D6E0`
- Rose Pink: `#E8B4C7`
- Soft Lilac: `#DCCEF9`
- Sage Green: `#D9E8D4`
- Butter Cream: `#FFF4CC`
- Petal Peach: `#F9E0D0`
- Soft White: `#FFFDFC`
- Warm Grey Text: `#5F5A5A`

## Fonts
### Excel-safe font options
- Title: Georgia Bold or Cambria Bold
- Headings: Aptos Semibold or Calibri Bold
- Body: Aptos, Calibri, or Arial

## Layout zones
### Banner
- Merge `A1:L2`
- Fill: blush pink
- Font: title font, size 18–22, white or warm grey text
- Add title: `Teacher Command Center Dashboard`

### Teacher profile cards
- `A3:C6`
- Fill: soft white
- Border: rose pink outline

### Class profile cards
- `D3:F6`
- Fill: soft white
- Border: sage green outline

### Reminder/notes block
- `G3:L6`
- Fill: butter cream
- Border: petal peach outline

## KPI card styling
Use `A8:C17` as 10 stacked KPI cards.

### Suggested style
- Labels in column B
- Values in column C
- Fill alternating between blush pink, lilac, sage, and cream
- Rounded look simulated with thick outside borders and generous padding
- Value cells bold, size 14–16

## Table block styling
### Upcoming dates `E8:H15`
- Header fill: rose pink
- Header text: white
- Body fill: soft white
- Alternate rows: very pale pink

### Attendance snapshot `J8:L15`
- Header fill: sage green
- Body fill: soft white

### Student support alerts `A20:D30`
- Header fill: lilac
- Below/intervention flagged rows: light peach fill

### Assignments and assessments `F20:I30`
- Header fill: butter cream
- Overdue rows: soft red fill
- Due soon rows: pale yellow fill

### Recent communication `J20:L30`
- Header fill: petal peach
- Body fill: soft white

## Footer sections
### Planning snapshot `A33:F40`
- Fill: soft white
- Border: lilac

### Professional snapshot `H33:L40`
- Fill: soft white
- Border: sage green

## Decorative touches
- Use one floral icon or flower emoji in the title only: `🌸`
- Keep the rest clean and minimal
- Avoid too many saturated colours
- Use thin borders inside tables and thicker borders around section cards

## Conditional formatting colours
- Overdue: `#F4CCCC`
- Due soon: `#FFF2CC`
- Completed: `#D9EAD3`
- Intervention flag: `#FCE5CD`
- Below expectation: `#F9CB9C`

## Final aesthetic goal
The workbook should feel:
- organized
- calm
- feminine but professional
- classroom friendly
- easy to scan quickly during a busy school day
